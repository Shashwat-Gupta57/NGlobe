# Security Policy

## Supported Versions

Currently, only the latest release of NGlobe is actively supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within NGlobe, please DO NOT open a public issue. 

Instead, please send an email to the repository maintainers. We will try our best to address the vulnerability promptly. We will acknowledge your email within 48 hours, and provide a timeline for a fix.
